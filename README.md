#  Student Management System (C++ Console Project)

## 📌 Overview
This is a simple **Student Management System** built using C++.  
It is a menu-driven console application that allows users to manage student records efficiently using **OOP concepts, vectors, and file handling**.

The system stores data permanently in a text file (`students.txt`) so that records are not lost after closing the program.

---

## 🚀 Features

- ➕ Add new student
- 📋 View all students
- 🔍 Search student by ID
- ❌ Delete student by ID
- 💾 Persistent storage using file handling
- 🔄 Auto-load data when program starts

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming (OOP)
- Vectors (STL)
- File Handling (`fstream`)
- Console-based UI
- GCC Compiler
